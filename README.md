# Truth Table Generator 🔥

A Java Spring Boot web application that generates truth tables for logical expressions.

## 🚀 Features
- Supports ∧, ∨, ¬, →, ↔ operators
- Handles complex expressions with brackets
- Dynamic truth table generation
- Postfix expression display
- Modern UI with dark mode
- Copy table feature

## 🛠️ Tech Stack
- Java (Spring Boot)
- HTML, CSS, Thymeleaf

## ▶️ How to Run
1. Open in IntelliJ IDEA
2. Run `TruthtableApplication.java`
3. Open http://localhost:8080

## 📌 Example Input
(P ∧ Q) → R
